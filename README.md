# Football Idle Game

A React-based idle football management game.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Project Structure

- `src/components/`: Reusable UI components
- `src/screens/`: Main screen components
- `src/hooks/`: Custom React hooks
- `src/store/`: State management with reducer and context
- `src/services/`: External services (storage, ads, IAP)
- `src/utils/`: Utility functions
- `src/constants/`: Game constants

## Troubleshooting

- If you encounter issues with dependencies, try deleting `node_modules` and running `npm install` again.
- Ensure you have Node.js version 14 or higher installed.
